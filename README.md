# Web Scraper

Scrapes web pages that contains posts and shows title, subtitle, published date and content

## Prerequisites
- OpenJDK ~1.80
- Apache Maven ~3.3.9

## Installation
```shell
git clone git@github.com:g1stavo/web-scraper ; cd web-scraper ; mvn dependency:resolve
```

## Usage
```shell
mvn exec:java
```
